## vue 실행 순서
###Vue.js 실전 연습 순서
vue --version

/* Vue.js 설치 안 되었을 때 */

npm i -g @vue/cli

vue --version

npm i -g @vue/cli-init
vue init webpack simple

/* simple을 프로젝트 이름이라고 생각하시면 됩니다.(폴더로 생성된다) */

cd simple
npm run dev
